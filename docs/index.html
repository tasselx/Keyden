Secure Authentication, Right in Your Menu Bar

Keyden is a lightweight, open-source TOTP authenticator designed exclusively for macOS. Keep your two-factor codes private and accessible.

Download for macOS View on GitHub

Why Choose Keyden?

Built with privacy, security, and simplicity in mind

Secure Storage

TOTP secrets stored in macOS Keychain with encryption. Your codes never leave your device.

One-Click Copy

Click to copy verification codes instantly. Access your codes from the menu bar anytime.

QR Code Support

Scan QR codes to add accounts or export tokens as QR images for backup.

Easy Migration

Import accounts from Google Authenticator via migration QR code or link. Batch import supported.

GitHub Gist Sync

Optional sync via private GitHub Gist. Works offline first, all data encrypted locally.

Multi-Language

English, Simplified Chinese, Traditional Chinese, and Japanese. Light/Dark mode supported.

Pin & Reorder

Pin frequently used accounts, drag to reorder. Group accounts by issuer for better organization.

Global Hotkey

Customizable keyboard shortcut (default: ⌘⇧K). Launch at login supported.

CLI Tool

Command-line interface for scripts and automation. Perfect for developers and power users.
